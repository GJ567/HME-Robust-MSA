# Top-k 相似样本加权增强方案

## 1. 原始 HME 的做法

原始 HME 在 `get_similar_embeddings` 中，对当前 batch 内样本计算余弦相似度。

对于每个样本，选择相似度超过阈值 `similarity_threshold` 的样本，然后直接取平均，得到增强表示。

## 2. 原始方法的问题

### 问题 1：相似样本直接平均

原始方法把所有超过阈值的样本同等看待。

但实际上，相似度 0.95 的样本应该比相似度 0.61 的样本更重要。

### 问题 2：阈值筛选不稳定

如果 batch 内刚好没有超过阈值的样本，就只能用平均表示兜底。

这会导致增强信息不够精准。

### 问题 3：只在 batch 内找

当前 batch 可能很小，不一定有真正相似的样本。

后续可以扩展到 memory bank 或情感记忆库。
